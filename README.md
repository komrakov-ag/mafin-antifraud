# MAFIN - Magnat Anti-Fraud Inspector

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

Пет система мониторинга мошенничества в реальном времени с фейковыми данными.

## 📊 Возможности

- **Дашборд** - метрики в реальном времени, графики трендов
- **Сессии** - мониторинг сессий пользователей с фильтрацией
- **Транзакции** - отслеживание транзакций, фильтры по сумме и дате
- **Алерты** - система оповещений, ручное/автоматическое решение
- **География** - тепловая карта IP, топ стран и городов
- **Отчеты** - формирование отчетов в PDF, CSV, Excel
- **Настройки** - настройка правил детекции

## 🛠 Технологии

- **Backend**: Go 1.21+
- **Frontend**: HTMX, Chart.js, Leaflet
- **Хранение**: In-memory (данные в памяти)
